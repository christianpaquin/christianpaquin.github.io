XPOC client-side pages

This is a collection of client-side pages that can be used to test the XPOC framework; these are available in the XPOC GitHub project.

  • Manifest editor: create and update XPOC manifests.
  • CSV-to-manifest converter: transform comma-separated-value files (as produced by Excel) into XPOC manifest.
  • Manifest viewer: inspect a XPOC manifest.
  • Content verifier: verify a XPOC resource.
